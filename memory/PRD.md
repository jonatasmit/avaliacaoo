# PRD - Página de Confiança Suplementos Mais Baratos

## Data de Criação: 2026-03-20
## Última Atualização: 2026-03-20

## Problem Statement Original
Construir uma página similar à sustanxt.suplementosmaisbaratos.com.br, mas focada em aumentar a confiança dos clientes usando mídias anexadas. O vídeo deve ter autoplay após primeira interação do usuário, música de fundo hospedada em avaliacoes.suplementosmaisbaratos.com.br.

## User Personas
- **Visitantes Céticos**: Pessoas que querem comprar mas não confiam no processo
- **Novos Clientes**: Pessoas que descobriram a loja e precisam de validação social
- **Clientes em Dúvida**: Pessoas que precisam de garantias antes de finalizar compra

## Core Requirements
1. ✅ Identidade visual consistente com sustanxt.suplementosmaisbaratos.com.br
2. ✅ Vídeo de avaliações com autoplay após primeira interação
3. ✅ Música de fundo após primeira interação do usuário
4. ✅ Informações de confiança: Frete Grátis, 24-48h envio, Garantia 7 dias
5. ✅ Produtos importados, 10+ anos de mercado, 21.000+ clientes
6. ✅ Todas as bandeiras de cartão aceitas
7. ✅ SEO e meta informações completas
8. ✅ Crisp Chat integrado (substituiu WhatsApp flutuante)
9. ✅ Menu com categorias de produtos (Injetáveis e Orais)
10. ✅ Rastreio via WhatsApp
11. ✅ CNPJ atualizado: Sucesso Vip Empreendimentos 21.534.683/0001-37
12. ✅ Mascote no background

## Architecture
- **Frontend**: React.js + TailwindCSS
- **Chat**: Crisp Chat
- **Estilo**: Dark theme profissional com efeitos glow e degradê
- **Cores**: Vermelho (#ff4d4d), Dourado (#ffd700), Rosa (#ff6b9d para Orais), Verde (#4ade80 para Avaliações)
- **Fonts**: Bebas Neue (display) + Montserrat (body)
- **Responsivo**: Mobile-first design
